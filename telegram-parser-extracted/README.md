# Telegram Car Parser

Автономный модуль парсера для интеграции в новые проекты. Полностью извлечен из исходного проекта Volga-Avto.

## Установка

1. Выполните `npm install`
2. Скопируйте `.env.example` в `.env` и заполните ваши доступы.
3. Запустите `npm start`

## Использование

Для настройки Webhook откройте или отправьте GET запрос на:
`http://<ВАШ_ДОМЕН>/api/tg-parser?setup=true`

Бот начнет автоматически принимать посты из каналов и личные сообщения, парсить информацию (характеристики, фото) и сохранять в базу данных Supabase (таблица `cars`).
