Сайт написан на Next.js

Поскольку сайт использует современные технологии (Next.js/React), он не может быть просто открыт через этот файл index.html.

Для того чтобы увидеть изменения, нужно запустить локальный сервер:
  1. Закройте этот файл.
  2. Найдите и запустите кликом файл Запуск_Сайта.bat, который находится в этой же папке.
  3. Подождите пару секунд, пока запустится сервер.
  4. Затем нажмите на кнопку ниже или введите в браузере адрес http://localhost:3000.
💻 Открыть сайт (localhost:3000)
