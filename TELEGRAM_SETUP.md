## Как получить Chat ID:

### Вариант 1: Личные сообщения (рекомендуется)
1. Найдите бота [@userinfobot](https://t.me/userinfobot) в Telegram
2. Напишите ему `/start`
3. Бот пришлёт ваш **Chat ID** (например: `123456789`)
4. Скопируйте это число

### Вариант 2: Отправка в канал
1. Создайте канал в Telegram (если ещё нет)
2. Добавьте вашего бота в канал как администратора
3. Chat ID канала будет вида: `-1001234567890`

## Как указать Chat ID в проекте:

1. Откройте файл `.env.local` в корне проекта
2. Найдите строку: `TELEGRAM_CHAT_ID=YOUR_CHAT_ID_HERE`
3. Замените `YOUR_CHAT_ID_HERE` на ваш Chat ID:
   ```
   TELEGRAM_CHAT_ID=123456789
   ```
4. Сохраните файл
5. Перезапустите сервер разработки (остановите и снова запустите `npm run dev`)

## Проверка работы:

1. Откройте ваш сайт
2. Заполните любую форму заявки
3. Нажмите "Оставить заявку"
4. Проверьте Telegram - должно прийти сообщение с данными заявки!

## Troubleshooting:

**Проблема**: Сообщения не приходят
- Проверьте, что Chat ID указан правильно в `.env.local`
- Убедитесь, что перезапустили сервер после изменения `.env.local`
- Проверьте, что бот не заблокирован в Telegram

**Проблема**: Ошибка "Telegram не настроен"
- Убедитесь, что в `.env.local` указан ваш Chat ID (не `YOUR_CHAT_ID_HERE`)
